I built a Book Recommendation System as part of my learning journey in Java and database integration.
🔹Features:
✅Add New Book
✅View All Books
✅Update Book By Name
✅Search Book, 
And many Features are available.
🔹Tech Stack: Core Java, JDBC, MySQL

This project helped me strengthen my knowledge of Java OOPS, JDBC connectivity, SQL queries, and exception handling, and also used build tool like Maven and Lombok is Library .It was a great hands-on experience in building a console-based real-world application.
